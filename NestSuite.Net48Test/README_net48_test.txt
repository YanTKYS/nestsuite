NestSuite .NET Framework 4.8 テストビルド
[検証終了 / 正式採用保留 / 正式サポート外]

このプロジェクトは v2.7.0〜v2.7.2 で実施した .NET Framework 4.8
互換性検証（net48_test）の成果物です。

v2.7.3 をもって検証を終了し、正式採用を保留としました。

----------------------------------------------------------------
検証結果（v2.7.3 時点）
----------------------------------------------------------------
- 実機起動: 確認済み
- 問題点: 複数 DLL を含む複数ファイル構成となる
- 判断: NestSuite の配布方針（単一EXE重視）に合致しないため
  正式採用しない

今後の正式配布は .NET 8 self-contained single-file 版を継続します。

----------------------------------------------------------------
正式配布版
----------------------------------------------------------------
正式リリースは nestsuite_*.zip（.NET 8 自己完結版）を使用してください。
単一EXEで動作し、ランタイムのインストールは不要です。

----------------------------------------------------------------
このプロジェクトについて
----------------------------------------------------------------
- 検証履歴として保持しています
- 通常ビルド・通常リリースからは参照されません
- net48_test の追加互換修正は原則停止です
- 単一EXE化の手段が確立された場合は再検討の余地があります
